Rock's Simple OpenPose Editor

Step 1. Select canvas size (output PNG image size).
Step 2. Select a built-in preset or load a JSON preset file that has been previously 'saved by this app'.
Step 3. Adjust the pose. (Note: Grab the 'joints', not the 'limbs'. You can also use the 'Shift control' to move the whole pose, up/down/left/right.)
Step 4. When finished adjusting the pose, save the PNG file for use with the OpenPose ControlNet.
